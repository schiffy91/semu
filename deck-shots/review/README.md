# Semu bezel review — Steam Deck captures (2026-06-30)

Folder layout: `<system>/<mode>/<variant>.jpg`
- modes: `game-priority` (game fills screen, bezel cropped), `bezel-priority` (whole device/TV visible, default), `alignment` (purple = declared screen hole, game @ 40% opacity — verifies the game-vs-bezel fit)
- `_overview/all-*.jpg` = montages across all systems

Systems captured: nes, snes, genesis, n64 (CRT-TV scenes); gb, gbc, gba (photoreal device shells).
Variants: nes(stock,black) snes/genesis(stock,solidA,solidB) gb(grey,red,green) gbc(teal,berry,grape) gba(indigo,sp=Game Boy Advance SP clamshell).

Not captured: psx (no PSX core installed on the Deck), nds (melonds needs DS firmware to boot).
